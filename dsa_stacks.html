Stacks & Queues

LIFO and FIFO data structures

Stack - Last In First Out (LIFO)

A stack is a linear data structure that follows the LIFO principle. The last element added is the first one to be removed, like a stack of plates.

Basic Operations:

  • Push: Add element to top - O(1)
  • Pop: Remove element from top - O(1)
  • Peek/Top: View top element - O(1)
  • isEmpty: Check if stack is empty - O(1)
// Stack implementation using array
class Stack {
    int top;
    int arr[100];
public:
    Stack() { top = -1; }
    
    void push(int x) {
        if (top >= 99) {
            cout << "Stack Overflow";
            return;
        }
        arr[++top] = x;
    }
    
    int pop() {
        if (top < 0) {
            cout << "Stack Underflow";
            return -1;
        }
        return arr[top--];
    }
    
    int peek() {
        return (top < 0) ? -1 : arr[top];
    }
    
    bool isEmpty() {
        return (top < 0);
    }
};

Queue - First In First Out (FIFO)

A queue follows the FIFO principle. The first element added is the first one to be removed, like a line of people waiting.

Basic Operations:

  • Enqueue: Add element to rear - O(1)
  • Dequeue: Remove element from front - O(1)
  • Front: View front element - O(1)
  • isEmpty: Check if queue is empty - O(1)
// Queue implementation using array
class Queue {
    int front, rear, size;
    int arr[100];
public:
    Queue() { front = rear = -1; size = 0; }
    
    void enqueue(int x) {
        if (size >= 100) {
            cout << "Queue Full";
            return;
        }
        if (front == -1) front = 0;
        arr[++rear] = x;
        size++;
    }
    
    int dequeue() {
        if (size == 0) {
            cout << "Queue Empty";
            return -1;
        }
        int x = arr[front++];
        size--;
        if (size == 0) front = rear = -1;
        return x;
    }
};

Applications

Stack Applications:

  • Function call management (call stack)
  • Expression evaluation and syntax parsing
  • Undo/Redo operations in editors
  • Backtracking algorithms (DFS, maze solving)
  • Browser history (back button)

Queue Applications:

  • CPU scheduling and disk scheduling
  • Breadth-First Search (BFS)
  • Printer queue management
  • Handling requests in web servers
  • Call center phone systems

Common Problems

1. Balanced Parentheses

bool isBalanced(string s) {
    stack<char> st;
    for (char c : s) {
        if (c == '(' || c == '{' || c == '[') {
            st.push(c);
        } else {
            if (st.empty()) return false;
            char top = st.top();
            st.pop();
            if ((c == ')' && top != '(') ||
                (c == '}' && top != '{') ||
                (c == ']' && top != '['))
                return false;
        }
    }
    return st.empty();
}

2. Implement Queue using Stacks

class QueueUsingStacks {
    stack<int> s1, s2;
public:
    void enqueue(int x) {
        s1.push(x);
    }
    
    int dequeue() {
        if (s2.empty()) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        }
        if (s2.empty()) return -1;
        int x = s2.top();
        s2.pop();
        return x;
    }
};
