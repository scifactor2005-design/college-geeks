Arrays & Strings

The foundation of data storage and manipulation

Introduction

Arrays and strings are the most fundamental data structures in computer science. An array is a collection of elements stored in contiguous memory locations, while a string is essentially an array of characters. Understanding these structures is crucial for efficient data manipulation and algorithm design.

Array Basics

An array stores elements of the same type in consecutive memory locations. This allows for constant-time access to any element using its index.

Key Characteristics:

  • Fixed Size: Traditional arrays have a fixed size determined at creation
  • Contiguous Memory: Elements are stored next to each other in memory
  • Index-Based Access: Elements can be accessed using their position (0-indexed)
  • Homogeneous: All elements must be of the same data type
// Array Declaration and Initialization
int arr[5] = {10, 20, 30, 40, 50};

// Accessing elements
int first = arr[0];  // 10
int third = arr[2];  // 30

// Modifying elements
arr[1] = 25;  // arr is now {10, 25, 30, 40, 50}

Time Complexity

Operation Time Complexity Description
Access O(1) Direct access using index
Search O(n) Linear search through elements
Insertion O(n) May require shifting elements
Deletion O(n) May require shifting elements

String Operations

Strings are sequences of characters. In many languages, strings are immutable, meaning they cannot be changed after creation. Common string operations include:

  • Concatenation: Joining two or more strings
  • Substring: Extracting a portion of a string
  • Comparison: Checking if strings are equal or comparing lexicographically
  • Search: Finding a character or substring within a string
// String operations in C++
string str1 = "Hello";
string str2 = "World";

// Concatenation
string result = str1 + " " + str2;  // "Hello World"

// Length
int len = str1.length();  // 5

// Substring
string sub = result.substr(0, 5);  // "Hello"

// Character access
char ch = str1[0];  // 'H'

Common Algorithms

1. Two Pointer Technique

Used for problems involving pairs or subarrays. Two pointers move through the array based on certain conditions.

// Example: Check if array is palindrome
bool isPalindrome(int arr[], int n) {
    int left = 0, right = n - 1;
    while (left < right) {
        if (arr[left] != arr[right])
            return false;
        left++;
        right--;
    }
    return true;
}

2. Sliding Window

Efficient technique for problems involving subarrays or substrings of a specific size.

// Example: Maximum sum of k consecutive elements
int maxSum(int arr[], int n, int k) {
    int maxSum = 0, windowSum = 0;
    
    // First window
    for (int i = 0; i < k; i++)
        windowSum += arr[i];
    maxSum = windowSum;
    
    // Slide the window
    for (int i = k; i < n; i++) {
        windowSum += arr[i] - arr[i - k];
        maxSum = max(maxSum, windowSum);
    }
    return maxSum;
}

3. Prefix Sum

Precompute cumulative sums for efficient range sum queries.

// Build prefix sum array
void buildPrefixSum(int arr[], int prefix[], int n) {
    prefix[0] = arr[0];
    for (int i = 1; i < n; i++)
        prefix[i] = prefix[i-1] + arr[i];
}

// Query sum from index l to r
int rangeSum(int prefix[], int l, int r) {
    if (l == 0) return prefix[r];
    return prefix[r] - prefix[l-1];
}
