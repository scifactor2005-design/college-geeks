Reinforcement Learning

Learning through interaction

Introduction

Reinforcement learning trains agents to make sequential decisions by rewarding desired behaviors and punishing undesired ones.

Key Concepts

  • Agent, Environment, State, Action
  • Reward, Policy, Value Function
  • Q-Learning, SARSA
  • Deep Q-Networks (DQN)

Applications

Game playing (AlphaGo), robotics, autonomous vehicles, resource management.
