Recurrent Neural Networks

Sequence modeling

Introduction

RNNs are designed for sequential data. They have memory that captures information about previous inputs in the sequence.

Variants

  • Vanilla RNN: Basic recurrent architecture
  • LSTM: Long Short-Term Memory - handles long dependencies
  • GRU: Gated Recurrent Unit - simplified LSTM

Applications

Language modeling, machine translation, speech recognition, time series forecasting, video analysis.
